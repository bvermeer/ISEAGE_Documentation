Overview

In this section you will download and install ESXi onto your server. ESXi is a bare-metal hypervisor that provide a lightweight framework for Virtual Machines to run on top of.

ESXi

ESXi is free (registration required) bare metal hypervisor that allows guest virtual machines to be run directly on the host server with little additional overhead. ESXi allows virtual networks to be built inside of it with multiple virtual machines.

Download ESXi

First, download the image for ESXi 5.5. Before the image can be downloaded, you must create a free account with VMware and log in. After logging in, download the ISO image for ESXi 5.5. Also note, near the top of the page is the license key that you will need to register ESXi with later.




Figure 1: Location of ESXi key


Install ESXi

After downloading the ESXi ISO image, install ESXi on your server (the default settings are fine). If you need further information on installing ESXi, follow the directions in Details on Installing ESXi.

After ESXi has been installed, you will be presented with a screen which looks similar to Figure 2.




Figure 2: ESXi Main Screen


Map Network Interfaces on ESXi

After installing ESXi, one of the first things that should be done is to map and label the physical network adapters (NICs) on the server. To do this we will individually hook up the physical NICs to a switch or router and see which interface shows as active in ESXi. First, press the F2 key to enter the ESXi configuration page. Enter your root password and then press Enter.




Figure 3: ESXi Password


Use the arrow keys to select Configure Management Network and then press enter.




Figure 4: Configure Management Network


Press enter to select Network Adapters. You will then be with a screen that looks similar to Figure 5.




Figure 5: ESXi Network Adapters


If any of the NICs are currently connected to anything, the status for that NIC will show up ad connected on this screen. One by one, hook up to NICs on the server to either a switch or a router. Exit out of the Network Adapters dialog box by pressing the Esc key and then re-enter the Network Adapters dialog box by pressing the Enter key. The Network Adapters dialog box should now show a different VMNIC as connected. Repeat this process for all of the NICs on the server and either write down the NIC mapping or label the NICs on the server. You will need this information later to properly connect the server.

Configuring ESXi

To configure ESXi, you must first download and install the vSphere Client (Windows only). To do this, on a separate Windows machine that is on the same network as the server, open a web browser and navigate to the web address given on the ESXi main screen (see Figure 2 for a reference of where to find the ESXi server IP address). If you receive a warning about the site's security certificate not being trusted, ignore the warning and proceed anyway. After navigating to the ESXi server IP address, you will see a screen similar to Figure 6.




Figure 6: ESXi Server Web Page


Click on the "Download vSphere Client" link to download the vSphere client installer from the ESXi server. After the vSphere client is downloaded, install it. After installing the vSphere client, run the program. You will be presented with a window similar to Figure 4.




Figure 7: vSphere client login window


Enter the IP address of your ESXi server (the same one used to download the ESXi client). The username is root, and the password is the password you chose when setting up the ESXi server. After logging into vSphere you should see a screen similar to Figure 8.




Figure 8: vSphere Client Main Window


After logging into the vSphere client, the first thing to do is to add the ESXi license key into vSphere.

ESXi License Key Registration

  • After logging into the vSphere client, go to the Configuration Tab



    • Figure 9: vSphere Configuration Tab


  • Next, click the Licensed Features link inside of the Software box on the left side of the screen.



    • Figure 10: vSphere License Features


  • Click the edit link in the upper right hand side of the screen



    • Figure 11: vSphere License Edit


  • Select the Assign a new license key to the host box



    • Figure 12: vSphere Enter Key


  • Enter your license key (this can be found here) in the box and click the OK button.
After setting up the license key on ESXi, you are now ready to set up the Virtual Network inside of ESXi.
